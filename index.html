About Me

Building the intersection of science & web

I build educational and scientific web applications that make complex chemistry, materials science, and scientific data accessible through interactive visualization.

My development approach centers on AI-assisted workflows — using tools like Claude, ChatGPT, and Gemini to rapidly prototype, then deeply customizing, debugging, testing, and refining every line of generated code to meet production standards.

Currently expanding into Blender Python scripting for scientific visualization, procedural modeling, and animation — bringing 3D storytelling to research and education.

My focus domains: chemistry, materials science, educational technology, and scientific visualization — building tools that researchers, students, and educators actually want to use.
